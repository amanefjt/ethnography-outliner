# タスク：カードのリサイズ機能と「マスク」表現の完成

カード（ノート）のリサイズ機能を実装し、折りたたみ時の挙動を「サイズ不変のマスク」として洗練させ、操作感の微調整を行う。

## 目的
- カードもグループ同様に端をドラッグして大きさを自由に変えられるようにする。
- 折りたたみ（名前表示）時に、カードやグループのサイズが変わらず、中身だけが中央のタイトルで「上書き（マスク）」される表現にする。
- コンテンツ表示が隠れてしまうバグ（グループ内のカード消失）を完全に修正する。
- トラックパッドでのズーム感度を落とし、二本指操作をより滑らかにする。

## 完了定義 (DoD)
- [ ] カード（ノート）の右下、右、下をドラッグしてサイズ変更ができる。
- [ ] カード内のテキストが溢れる場合は、フォントサイズを変えずに `overflow: hidden` で省略される。
- [ ] 折りたたみボタンを押してもサイズ（width/height）は変わらず、中身が消えて中央にタイトルが表示される。
- [ ] グループ内のカードがレンダリングエラーや重なりによって消失せず、正しく表示される。
- [ ] マウスホイール（トラックパッド）でのズーム速度が現在の約半分になり、制御しやすくなる。

## ステップ
1. [ ] `main.js`: `CONSTANTS.ZOOM_STEP_WHEEL` を小さく調整。
2. [ ] `main.js`: `startResize`, `handleResize` をノートカードにも適用されるよう修正。
3. [ ] `style.css`: `.collapsed` 時のスタイルを「サイズ固定・中央揃え」に刷新。
    - `width: 100%`, `height: 100%` のオーバーレイ構造を検討。
4. [ ] `style.css`: `.note-card` に `overflow: hidden` を追加。
5. [ ] 動作確認。
