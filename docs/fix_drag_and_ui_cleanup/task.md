# タスク：ドラッグ＆ドロップの修正とUIの全体調整

ドラッグ＆ドロップ機能が動作しなくなっているバグを修正し、UI全体のレイアウトとコードの整合性を整える。

## 目的
- 壊れているコア機能（ドラッグ＆ドロップ）を復旧させる。
- UIから不要な要素を削除し、デザイン要素を洗練させる。
- コードの重複や不整合を解消し、メンテナンス性を向上させる。

## 完了定義 (DoD)
- [ ] ノートおよびグループをキャンバス上で自由にドラッグ＆ドロップできる。
- [ ] グループを移動した際、その中のノートやサブグループも連動して移動する。
- [ ] カード内のID表示が目立たなくなっている（または非表示）。
- [ ] 不要なリンク（鎖）アイコンがコードおよび画面から完全に削除されている。
- [ ] リサイズ機能が正しく動作し、境界の自動更新が機能している。
- [ ] `main.js` 内の重複したイベントリスナーや未使用のコードが整理されている。

## ステップ
1. [ ] `main.js` の `drag` 関数における `state.dragItem` の扱いを修正（バグ修正）。
2. [ ] `index.html` および `main.js` からリンク（鎖）アイコンのコードを削除。
3. [ ] `style.css` を調整し、カードのID表示を抑制、全体的な陰影や質感をプレミアムなものに更新。
4. [ ] `main.js` の整理（重複箇所の削除、一貫性のある変数名の使用）。
5. [ ] 動作確認。
