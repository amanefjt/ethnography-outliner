# タスク：グループ隠蔽の強化とテキストインポート機能の実装

グループの「名前のみ表示」時に中身を完全に非表示にする修正と、外部テキストを段落ごとにカード化して取り込む「インポート機能」を実装する。

## 目的
- グループを折りたたんだ際、単に背景で覆うのではなく、中のカード自体を非表示にして「情報のカプセル化」を徹底する。
- 長文のフィールドノートを、段落（改行）を区切りとして一括でカード化できるようにし、分析の準備を効率化する。

## 完了定義 (DoD)
- [ ] グループを折りたたむと、その中のすべてのノートおよびサブグループが（それらが展開状態であっても）キャンバス上から消える。
- [ ] テキストを貼り付けるためのインポート用モーダル、または入力エリアがある。
- [ ] 貼り付けたテキストを処理し、空行で区切られた各段落を一括で新規カードとして生成する。
- [ ] 生成されたカードは重ならないようにスマートに配置される。

## ステップ
1. [ ] `main.js`: `renderNotes` と `renderGroups` を修正し、親グループが `collapsed: true` の場合は描画をスキップ（または `display: none` を適用）するロジックを追加。
2. [ ] `index.html`: テキストインポート用のボタンとモーダル（textarea を含む）を追加。
3. [ ] `main.js`: インポート処理ロジックの実装（テキスト分割、ループでのカード生成）。
4. [ ] `style.css`: インポート画面のデザイン。
5. [ ] 動作確認。
