# Walkthrough - グループ・カード表示および操作性の改善

## Changes

### Logic Layer
- **グループ連動移動**: グループドラッグ時に、その下位階層にあるアイテム（ノート・子グループ）を再帰的に特定し、同時に座標移動させるロジックを追加しました。
- **スマイル配置アルゴリズム**: 新規カード/グループ作成時、ビューの中心から外側へ渦巻き状に空きスペースを探索し、重ならない位置に配置するようにしました。
- **折りたたみ機能**: ノートとグループに `collapsed` 状態を追加し、UIでの切り替えを可能にしました。グループ折りたたみ時は子要素を非表示にします。

### UI Layer
- **ID表示**: 極小サイズかつ薄い色に変更し、視覚的なノイズを減らしました。
- **リンクアイコン**: ノートのUIから削除しました。
- **折りたたみスタイル**: 
    - ノート: タイトルのみ表示。
    - グループ: ヘッダーのみ表示し、境界線を非表示/縮小。
