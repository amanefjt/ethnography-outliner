# タスク：折りたたみ機能の修正とアウトライナーの並び替え実装

折りたたみ機能のバグ（ヘッダー消失）を修正し、アウトライナービューで項目を上下にドラッグ＆ドロップで並び替えられるようにする。

## 目的
- 折りたたんだ際もヘッダー（タイトルや展開ボタン）が正しく表示され、元の状態に戻せるようにする。
- アウトライナービューで、Workflowyのように項目（グループやノート）を上下に並び替えられるようにする。
- アウトライナーでの並び替えがキャンバスの配置に影響を与えないようにする。

## 完了定義 (DoD)
- [ ] グループやノートを折りたたんでも、タイトルと展開ボタンが消えずに表示され続け、再クリックで展開できる。
- [ ] 折りたたみ時のデザインが適切で、操作不能にならない。
- [ ] アウトライナービューの項目がドラッグ＆ドロップで上下に移動できる。
- [ ] 並び替えた順序が `localStorage` に保存され、リロード後も維持される。
- [ ] アウトライナーでの順序変更が、キャンバス上の座標には影響しない。

## ステップ
1. [ ] `style.css`: `.group-header` の配置を修正し、`overflow: hidden` で消えないようにする。
2. [ ] `main.js`: `renderOutline` および `createOutlineItem` を更新し、ドラッグ＆ドロップ用イベントリスナーを追加。
3. [ ] `main.js`: `state.outlineOrder` を活用し、階層ごとの並び順を管理・保存するロジックを実装。
4. [ ] 動作確認。
