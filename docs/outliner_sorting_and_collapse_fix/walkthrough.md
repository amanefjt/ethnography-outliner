# ウォークスルー：折りたたみ機能の修正とアウトライナーの並び替え実装

## 修正内容の詳細

### 1. グループヘッダーの再配置 (`style.css` & `main.js`)
- グループカードのヘッダーを「枠の上」から「枠の内側の最上部」に移動しました。
- 折りたたみ時 (`.collapsed`) も、ヘッダー部分だけは固定の高さで表示されるように CSS を調整。
- これにより、折りたたんだままでもタイトルの編集や再展開が確実にできるようになりました。

### 2. アウトライナーでの並び替え機能
- アウトライナーの各項目（グループ、ノート）の左側にドラッグ用のハンドル、または項目自体をドラッグ可能にしました。
- ドラッグ中の項目がどこに挿入されるかを示す視覚的なインジケータ（青い線）を追加。
- ドロップ時に `state.outlineOrder` を更新し、`localStorage` に即座に保存するようにしました。

### 3. キャンバスとアウトラインの独立
- アウトライナーで順序をどれだけ入れ替えても、キャンバス上の座標（X, Y）は書き換えられません。
- 「空間的な自由配置（キャンバス）」と「論理的な構造化（アウトライン）」の両立を実現しました。

## 動作確認手順
1. キャンバスでグループの折りたたみボタン（^）を押す。ヘッダー（タイトル）が残り、中身だけが消えることを確認。
2. もう一度ボタンを押して元に戻ることを確認。
3. アウトライナービューに切り替える。
4. 「グループ1」をドラッグして「グループ2」の下に移動させる。順番が入れ替わることを確認。
5. ページをリロードしてもアウトライナーの順番が維持されていることを確認。

## 想定される副作用
- アウトライナーで新しく作成したアイテムは、デフォルトでリストの最後に追加されます。
