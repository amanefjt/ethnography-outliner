# Mission: 野外科学と喚起的なインターフェース

## 1. 中核となる哲学：野外科学 (Wilderness Science)
このプロジェクトは単なるソフトウェアツールではありません。これは**野外科学**のための道具です。
科学は実験室の中だけに存在するものではありません。混沌とした現実の真っ只中、「野外」において生まれる「態度」こそが科学です。
**Ethnography Outliner** は、この態度、すなわち「ありのままの世界に対する厳密かつ開かれた関わり」を支えるために存在します。

### 問題解決の「W型モデル」
川喜田二郎のKJ法に従い、私たちは人間の行為の「W型モデル」を遵守します：
1.  **探検 (Fieldwork):** 生のデータを収集し、混沌を受け入れる。（**キャンバスビュー**が支援）
2.  **概念化 (Abduction):** グループ化し、構造化し、隠れたパターンを見出す。（**KJ操作**が支援）
3.  **叙述 (Realization):** 構造を一貫した物語として表現する。（**アウトライナービュー**が支援）

## 2. ゴール：喚起的な民族誌 (Evocative Ethnography)
私たちの成果物は単なる報告書ではありません。それは**喚起的な民族誌**です。。
- **意味や関係性の断層:** 文化を一枚岩として描かず、その内部にある「矛盾」や「対立する真実」を露呈させます。
- **響き合うイメージ:** 映画『Forest of Bliss』（ロバート・ガードナー）のように、データ同士の並置（モンタージュ）によって意味を「響き合わせ（Resound）」ます。


## 3. アブダクションの重視
従来のツールはトップダウンの論理（演繹法）を強制します。このツールは**アブダクション（仮説形成）**を育成します。
- **データに語らせる:** ボトムアップで整理します。個別の観察が集まり、より広い概念を形成するようにします。
- **曖昧さの許容:** 不確かなつながりを保存します。「ゴーストノート」は、居場所が見つかるまでアイデアを浮遊させます。
- **空間的思考:** **キャンバス**は新しいつながりを発見する場所です。**アウトライナー**はそれを固める場所です。
