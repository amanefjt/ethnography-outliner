# タスク：折りたたみ機能のリファインと「マスク」表現の実装

グループとノートの折りたたみ（隠蔽）機能を、情報を一時的に覆い隠すような「マスク」的な振る舞いに変更し、視覚的な一貫性を高める。

## 目的
- 折りたたみ時、グループ名は中央に表示されるようにする。
- ノート（カード）も名前（タイトル）のみを表示できる機能を確認・安定化する。
- 折りたたみの挙動を「サイズ変更」というより「コンテンツの隠蔽」としてデザインし、ヘッダーが踊るバグを解消する。

## 完了定義 (DoD)
- [ ] グループを折りたたんだ際、高さが最小化され、タイトルがその領域の中央に水平・垂直配置される。
- [ ] カード（ノート）を折りたたんだ際、タイトルバーのみが表示され、本文が完全に隠れる。
- [ ] 折りたたみ・展開を繰り返しても、カードやヘッダーの位置が不自然にジャンプしない。
- [ ] 折りたたみ時のボタン（^ / ⌄）が常に正しく機能し、反転する。

## ステップ
1. [ ] `style.css`: `.collapsed` 時のスタイルを刷新。
    - グループ: `display: flex; align-items: center; justify-content: center;` を活用して中央揃えを実現。
    - ノート: ヘッダーのみを残し、コンテンツ高さを固定化。
2. [ ] `main.js`: `createGroupElement` および `createNoteElement` の HTML 構造を、中央揃えが容易なように調整。
3. [ ] `main.js`: トグルボタンのアイコン（Lucide）が状態に応じて「上向き/下向き」に切り替わるように修正。
4. [ ] 動作確認。
