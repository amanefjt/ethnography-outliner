# ウォークスルー：ビジュアルの強化と表示バグの修正

## 修正内容の詳細

### 1. 配色システムの刷新 (`style.css`)
- 各カラーバリエーションに対して 2 色（枠線用と背景用）を定義しました。
- ノートカードとグループカードの両方が、選択した色で塗りつぶされるようになりました。
- テキストの可読性を保つため、背景色は非常に淡い色（透明度を考慮した色）に設定しています。

### 2. 選択状態の視覚強調
- カード選択時の枠線を `1px` から `3.5px` に太くしました。
- 枠線の色を「琥珀色（アンバー）」に変更し、どの色が塗られているカードでも「選択中」であることが一目でわかるようにしました。

### 3. グループヘッダーの修正
- グループ名の表示部分を `position: absolute` に変更し、カードの最上部に張り付くように固定しました。
- ドラッグ中やリサイズ中もヘッダーが踊ったり消えたりすることなく、常にカードの上端に位置し続けます。
- 折り畳み時のデザインもこの修正に合わせて再構成し、コンパクトかつ操作性を維持した外観に整えました。

## 動作確認手順
1. カードまたはグループのパレットアイコンをクリックして色を変更する。枠だけでなく全体の色が淡く変わることを確認。
2. そのアイテムをクリックして選択する。枠線が太い琥珀色に変わることを確認。
3. グループをドラッグして動かす。名前部分がカードの上端から離れずに追従することを確認。
4. グループを折り畳む。ヘッダー（タイトル）が正しく表示されていることを確認。

## 想定される副作用
- 全体的に色がつくため、多数のカードが重なっている場合に境界が少し見にくく感じる可能性があります（枠線の太さを強めることで対策済み）。
